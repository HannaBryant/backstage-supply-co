# PROJECT OVERVIEW

Backstage Supply Co. is my final capstone project completed as part of the Codex Academy Full-Stack Development program. The project combines frontend development, backend development, database management, cloud deployment, and workflow automation into a single business application.

The application simulates an inventory management platform for a touring entertainment and merchandise company. Users can manage product records, track inventory quantities, update pricing information, and monitor available stock through a centralized web interface.

The frontend was developed using React and Vite, providing a responsive user experience and dynamic data updates. The backend was built with Flask and Python and exposes RESTful API endpoints responsible for handling product operations. Product information is stored in a PostgreSQL database using the backstage_supply schema.

A key component of this capstone project is the integration of an AI-powered assistant using n8n automation workflows. The workflow connects application data with AI capabilities, demonstrating how automation and artificial intelligence can be integrated into modern business systems.

The completed application was deployed to AWS, creating a production-ready solution that showcases the full software development lifecycle from planning and database design through deployment and automation.
